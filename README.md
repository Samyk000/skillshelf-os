# 🎨 Skillshelf Open Design

The open-source registry of AI-ready design systems. Drop a `SKILL.md` into your project and give your AI coding agent pixel-perfect design DNA.

[![Skills](https://img.shields.io/badge/skills-1-blue)](https://github.com/Samyk000/skillshelf-OpenDesign/tree/main/skills)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)

## 🔥 Why This Exists
AI coding agents are incredible at logic, terrible at design. They generate generic, inconsistent UI. This registry fixes that. Each skill is a handcrafted design system that teaches AI exactly how to build beautiful, consistent interfaces.

## 📦 Quick Start
1. Browse the [skills directory](/skills/)
2. Copy a `SKILL.md` into your project root
3. Tell your AI: `"Follow the design system in SKILL.md"`
4. Watch it generate pixel-perfect UI

## 🤝 Contribute
Missing a design system? We accept PRs!
1. Fork the repo
2. Add a new folder to `skills/[brand]/`
3. Include `SKILL.md`, `preview.html`, and `cover.png`
4. Update `registry.json`
5. Open a PR

See [CONTRIBUTING.md](CONTRIBUTING.md) for templates and guidelines.

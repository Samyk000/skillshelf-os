# Security Policy

## Supported Versions

We release updates to the registry as new skills are added or existing ones are improved.

| Version | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |

## Reporting a Vulnerability

While this repository contains design specifications and not executable code, we take the integrity of our content seriously.

If you find a security issue related to:
- Malicious content in skill files
- Broken or harmful preview HTML
- Registry data integrity issues

Please report it by opening a private issue or contacting us directly at [your-email@example.com].

We will respond within 48 hours.

## Content Safety

All preview HTML files are intended for local development use only. They should not contain external tracking scripts or malicious code. Contributors must ensure their submissions are clean and safe.
